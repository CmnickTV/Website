# CMNickTV Website

A free GitHub Pages website for CMNickTV.

## How to publish on GitHub Pages

1. Go to https://github.com/new
2. Create a new repository named `cmnicktv.github.io`
3. Upload these files:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
4. Open the repository Settings
5. Go to Pages
6. Under Build and deployment, choose Deploy from a branch
7. Select the `main` branch and `/root`
8. Save

Your site will publish at:

https://your-github-username.github.io/

## Edit links

Open `index.html` and replace the Twitch, YouTube, Bluesky, and Whatnot links with your real links.
